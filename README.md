# GraphQL Todo List

## Project Overview
### Client
- Vue CLI 3
- vue-cli-plugin-apollo

### Server
- express
- apollo-server-express


## Project setup
```
npm install
```

### 1. Start back-end server
```
npm run server
```

### 2. Start front-end server
```
npm run serve
```

### 3. Open browser
```
http://localhost:8080/
```
